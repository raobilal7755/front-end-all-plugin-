Accordion (Slide effect, Multilevel)

Slide effect initialization:

Script usage

There are the following options in the script:

		jQuery('ul.accordion').slideAccordion({
			activeClass:'open',      // active class for the open state
			opener:'>a.opener',      // opener selector
			slider:'>div.slide',     // accordion's drop-down selector
			collapsible:false,       // allow to close accordion's active element
			animSpeed: 300           // accordion's animation speed
		});

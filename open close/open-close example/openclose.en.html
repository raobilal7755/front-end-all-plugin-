Slide effect initialization:

Script usage

There are the following options in the script:

		jQuery('div.toggle-block').openClose({
			addClassBeforeAnimation: true,     // change "active" class before the navigation
			activeClass:'active',              // active class for open state
			opener:'.opener',                  // link opener selector
			slider:'.slide',                   // selector of the sliding block
			animSpeed: 400,                    // animation speed (400ms = 0,4s)
			effect:'fade',                     // effect "fade", "slide" or "none"
			event:'click'                      // event for action
		});

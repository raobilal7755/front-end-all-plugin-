Add class on click

Example 1. Add class when clicking on <div>.

			jQuery('div.div-class').clickClass({
				classAdd: 'div-add-class'
			});
<div> for click

Example 2. Add class on parent when clicking on <a>

			jQuery('a.a-class').clickClass({
				classAdd: 'div-parent-add-class',
				addToParent: true
			});
<a> for click
